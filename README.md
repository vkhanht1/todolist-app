# 📝 Modern To-Do List Application

A stylish and functional Task Management web application built with a modern **Glassmorphism UI**. This project demonstrates clean frontend development practices, focusing on interactive user experience and persistent data storage.

## ✨ Key Features
* **Interactive Task Creation:** Add new tasks quickly with an intuitive input system.
* **Task Management:** Mark tasks as completed or delete them with a single click.
* **Glassmorphism UI:** A beautiful, modern frosted-glass effect background for a premium look.
* **Responsive Design:** Optimized for various screen sizes, ensuring a great experience on mobile and desktop.
* **Local Storage:** Your tasks are saved locally in the browser, so they persist even after a page refresh.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for web content.
* **CSS3:** Advanced styling including Flexbox, Grid, and Backdrop-filter for the Glassmorphism effect.
* **JavaScript (ES6+):** Core logic for task manipulation and event handling.
